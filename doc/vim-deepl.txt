===============================================================================
INTRODUCTION                                                         *deepl-trainer*

deepl-trainer is a Vim plugin that integrates DeepL translation directly into Vim.
It allows you to:
- Translate single words or short phrases (up to 3 words)
- Build a personal dictionary automatically
- Practice vocabulary inside Vim using a spaced repetition trainer
- Translate longer selections into a bottom history window
- Fully async (Vim 8 / job_start / channels)
- Works offline with dictionary cache after first translation

===========================================================================

INSTALLATION                                               	     *deepl-trainer*

Using vim-plug:

    Plug 'Romariozh/vim-deepl'

Then run:

    :PlugInstall

Make sure you have a DeepL API key:

    export DEEPL_API_KEY="your_token"

Python 3 required.

VOCABULARY TRAINER                                                   *deepl-trainer*

Command:

    :DeepLTrainerStart

Trainer buffer keys:

    n   Next item (asynchronous)
    s   Show translation
    x   Mark ‚Äúdon‚Äôt know‚Äù (increase difficulty)
    d   Never show again (ignored in future)
    q   Quit trainer window

Trainer features:
- Spaced repetition based on count / difficulty / last_seen
- Items sorted by weakest first
- Shows global mastery progress

===========================================================================

HISTORY WINDOW                                                       *deepl-trainer*

Displayed when translating selections of 4+ words.

Behavior:
- Opens at bottom with fixed 5-line height
- Appends each translation with timestamp and language info
- Auto-scrolls to bottom
- Focus returns to original editing window

Buffer mapping:

    q   Clear history and close the window

CONFIGURATION OPTIONS                              		     *deepl-trainer*

| Variable                   | Default                                         |
|---------------------------|-------------------------------------------------|
| g:deepl_api_key           | $DEEPL_API_KEY                                   |
| g:deepl_target_lang       | 'RU'                                             |
| g:deepl_lang_cycle        | ['RU', 'EN', 'DA']                               |
| g:deepl_word_src_lang     | 'EN'                                             |
| g:deepl_word_src_cycle    | ['EN', 'DA']                                     |
| g:deepl_helper_path       | plugin_root/python/deepl_helper.py               |
| g:deepl_dict_path_base    | ~/.local/share/vim-deepl/dict            |

Dictionary files example:

    ~/.local/share/vim-deepl/dict_en.json
    ~/.local/share/vim-deepl/dict_da.json

STATUSLINE SUPPORT                                    *deepl-trainer-statusline*

    let g:lightline = {
    \ 'component_function': {
    \   'deepl': 'DeepLStatus',
    \ },
    \ }

Shows:

    [DL:RU]
